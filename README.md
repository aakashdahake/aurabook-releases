# AuraBook — Releases
This repository hosts the public download releases for **AuraBook**, a business management desktop app for macOS.
## Download
Go to the [**Releases**](https://github.com/aakashdahake/aurabook-releases/releases) page to download the latest version.
## Installation
1. Download the `.dmg` file for your Mac
2. Open the DMG and drag **AuraBook** into your Applications folder
3. Launch AuraBook from Applications
**System Requirements:** macOS 12 (Monterey) or later · Apple Silicon (ARM64)
